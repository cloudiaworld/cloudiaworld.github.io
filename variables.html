Connectivity

LoRa

The LoRa technology allows building up scalable networks across long distances with very low, near zero operational costs.

WiFi

Private users, as well as users requiring higher data throughput can benefit from the built-in WiFi functionality.

Bluetooth

It is primarily available for configuration, but can be used for transmitting data over short distances.

GSM

There are some applications and places in which GSM is the only available solution. CLOUDia incorporates a second rack for a battery in order to support GSM.
